# 2023 Hackathon — Embu

Site for the 2023 Hackathon Workshop on "Computational Modeling to Unleash the Power of Young African Researchers", held at the University of Embu, Kenya. SvelteKit 5, adapter-static. Deployed at `/2023-hackathon-embu`.

Styling and shared UI come from the lab [`@computational-biology-aachen/design`](https://github.com/Computational-Biology-Aachen/design) system (design tokens + Svelte components).

## Dev

```bash
npm install
npm run dev          # dev server on :5173
npm run build        # static build → build/
npm run preview      # preview the build
npm run check        # TypeScript + Svelte type checking
npm run lint         # Prettier + ESLint check
npm run format       # auto-format with Prettier
```
